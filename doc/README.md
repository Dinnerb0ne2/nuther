## 🎯 核心概念

### LSTM (Long Short-Term Memory)
长短期记忆网络，一种特殊的 RNN，能够学习长期依赖关系。

### 记忆回溯 (Retro Memory)
通过检索相关记忆来增强上下文理解，支持关键词加权匹配。

### MoE (Mixture of Experts)
混合专家模型，通过门控机制选择合适的专家进行处理。

### 对话式神经网络
能够进行多轮对话、理解上下文的神经网络模型。
